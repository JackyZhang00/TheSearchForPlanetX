# TheSearchForPlanetX
《X行星之谜》-- 文字版

作者：Jacky Zhang

## 软件介绍

本软件主要是针对于桌游《X行星之谜》所设计的纯文字版。理论上该程序**预计**可以实现本地多人游戏。但考虑到实际桌游的形式，仍然仅建议采用桌游的solo模式

*注：在采用solo模式时，请配备原版桌游配套app，[配套网址](https://www.PlanetXApp.com)*

## 软件使用

请直接执行代码`python main.py`即可。具体代码与操作可在运行时查看
